# Chatbot_experiment

* This is a simple chatbot project Using FLASK backend ready for depolyment (Dockerized)

* Integrated with Neo4j for Graph RAG

* Use of LLM API

```plaintext
│   └── utils.py
├── run.py   
├── requirements.txt
├── Dockerfile
├── .dockerignore
└── README.md
```
